# S&P 500 Stock Comparison Tool

Excel-based stock comparison dashboard analyzing AAPL, MSFT, and GOOGL over 500+ trading days.

## What It Does

Compares 3 major tech stocks using 2 years of daily price data from Yahoo Finance.

## Metrics Calculated (All Excel Formulas)

- Start/End Price
- Total Return & Annualized Return
- Daily & Annual Volatility (STDEV)
- Sharpe Ratio
- Win Rate (COUNTIF)
- Risk/Return Ratio
- Max Drawdown
- Pearson Correlation Matrix

## Charts

- Stock Price Comparison (line)
- Total Return Comparison (bar)
- Sharpe Ratio Comparison (bar)
- Volatility Comparison (bar)

## Key Finding

AAPL and MSFT showed near-zero correlation (-0.04) despite both being tech stocks, suggesting sector classification alone does not guarantee co-movement.

## Tools Used

- Excel (formulas, charts, data visualization)
- Yahoo Finance (data source)
